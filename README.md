# Resume Builder

A simple and powerful resume builder that helps you create clean, professional resumes in minutes — without paying for premium tools.

This project is built to make resume creation आसान for everyone. Most online builders lock good features behind paywalls, so the idea here is to keep everything free and actually useful.

---

## 🔗 Live Demo

👉 [https://moin-shadab.github.io/resume-builder/](https://moin-shadab.github.io/free-resume/)

---

## ✨ Features

* Easy-to-use interface
* Clean and professional layout
* Multiple resume sections (Education, Skills, Experience, etc.)
* Real-time preview
* Download/print resume
* Lightweight and fast

---

* Modern Resume
* Minimal Resume

---

## 📌 Why this project?

Most resume builders:

* Charge money for downloads
* Limit templates
* Add watermarks

This project tries to solve that by being:

* 100% free
* Open-source
* Simple and practical

---

## 🛠 How to use

1. Clone the repo
2. Open `index.html` in your browser
3. Start building your resume
